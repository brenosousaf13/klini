Dúvidas

Perguntas frequentes

Por que o token é necessário?

O token garante que o atendimento está sendo autorizado presencialmente, reduzindo fraudes e trazendo mais segurança para você.

O token expira?

Sim, por segurança o código é renovado a cada 60 segundos no aplicativo. Caso peça via WhatsApp, ele possui uma validade específica para aquele atendimento.

Posso usar o token sem internet?

O aplicativo precisa de internet para gerar novos tokens. Se você estiver sem conexão antes do atendimento, recomendamos salvar o código ou pedir via WhatsApp.

O que fazer se eu perder meu celular?

Entre em contato imediatamente com o nosso SAC (0800 021 0320) para bloquear o acesso ao seu aplicativo e receber orientações.
